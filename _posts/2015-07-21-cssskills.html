原创文章 转载注明出处

强制不换行，超出部分用省略号显示

    {
            overflow:hidden;
            white-space:nowrap;
            text-overflow:ellipsis;
    }

文本强制超出就换行，破坏长串字母和数字等的不法输入

    {
            word-wrap: break-word;
    }

设置图片最大宽度和最大高度不超过500px，图片不失真

    <img src='test.jpg' style="max-width: 500px; max-height: 500px;">

文本不可选中，兼容写法

    body,canvas,div{
        -moz-user-select:none;
        -webkit-user-select: none;
        -ms-user-select: none;
        -khtml-user-select:none;
        -webkit-tap-highlight-color: rgba(0,0,0,0);
    }

定位用法：

  1. position: relative; 不会浮动元素
  2. position是相对于递归上级父元素设置了absolute和relative定位的，如果一直没有，就会相对文档定位。

width无效用法

width对display:inline使用无效，也因此，span天生不能用width约束。

两种盒子模型

标准W3C盒子模型

IE盒子模型

box-flex

一个例子:

CSS

    .test_box {
        display: -moz-box;
        display: -webkit-box;
        display: box;
        ...
    }
    .list { ... }
    .list_one {
        -moz-box-flex: 1;
        -webkit-box-flex: 1;
        box-flex: 1;
    }
    .list_two{
        -moz-box-flex: 2;
        -webkit-box-flex: 2;
        box-flex: 2;
    }

HTML

    <div class="test_box">
        <div class="list list_two">1</div>
        <div class="list list_one">2</div>
        <div class="list list_one">3</div>
    </div>
