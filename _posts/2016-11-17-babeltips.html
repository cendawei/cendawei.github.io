原创文章 转载注明出处

1、vue用babel转码的时候，如果放心不下或转码不成功的话

可以这么做：

在webpack配置的根对象中，添加

    vue: {
         loaders: {
              js: babel-loader
         }
    }

注：同理css、less等也可以指定一个loader

2、babel打包报一些IE没定义等的全局变量缺失的错

方案：一般剔除第三方js包的babel打包

exclude: /(node_modules|lib)/

注意：正则表达式中间不允许出现空格

3、babel默认不转码的一些es6的API，eg. Promise

方案：安装babel-plugin-transform-runtime

Installation

    $ npm install babel-plugin-transform-runtime

Usage

.babelrc

    {
      "plugins": ["transform-runtime"]
    }
